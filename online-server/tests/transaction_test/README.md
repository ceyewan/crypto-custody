# 交易接口测试

该目录包含交易准备、签名发送、余额查询和交易列表相关的接口测试。测试会向本地 `http://localhost:8080` 发送请求，并可能连接测试网络。

## 文件说明

- `common_test.go`：测试辅助函数、登录工具和交易测试参数。
- `transaction_test.go`：交易查询和列表相关测试。
- `transfer_test.go`：完整转账流程测试。

## 运行前准备

1. 启动本地服务，并确认监听 `8080` 端口。
2. 设置警员账号密码环境变量：

```bash
export OFFICER_PASSWORD="your_officer_password"
```

3. 在 `common_test.go` 中确认测试地址和签名参数：

- `TestAccountA`：发送方以太坊地址。
- `TestAccountB`：接收方以太坊地址。
- `TestSignature`：与待签名消息匹配的签名。

## 运行方式

```bash
cd tests/transaction_test
go test -v
```

## 覆盖范围

- 查询以太坊地址余额。
- 准备交易并生成消息哈希。
- 提交签名并发送交易。
- 等待交易确认。
- 查询交易列表、详情和统计数据。

## 注意事项

- 交易测试可能真实广播测试网交易，请只使用测试网络账户和测试资产。
- 交易确认需要等待区块链出块，测试耗时会比普通接口测试更长。
- 警员账号需要预先存在，并拥有交易操作权限。
