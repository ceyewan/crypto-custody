# docs 目录

该目录保存面向开发和联调的说明文档。

## 文件说明

- `user_management_api.md`：用户管理接口说明。
- `user_management_guide.md`：用户管理模块开发指南。
- `transaction_management_api.md`：交易管理接口说明。
- `transaction_management_guide.md`：交易管理模块开发指南。

根目录的 `API_DOCUMENTATION.md` 是汇总版接口文档，适合快速查阅；本目录文档适合查看模块细节。

## 维护建议

- 修改接口路径、请求字段、响应字段或权限规则后，同步更新对应 API 文档。
- 修改业务流程后，同步更新对应 guide 文档。
- 示例请求应尽量保持可直接复制运行。
