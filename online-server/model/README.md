# model 目录

`model` 目录定义数据库模型，由 GORM 用于建表、迁移和数据读写。

## 文件说明

- `user_model.go`：用户账号、角色、密码哈希和状态字段。
- `account_model.go`：托管账户地址、所属用户和账户元数据。
- `transaction_model.go`：交易金额、地址、哈希、签名、状态和链上结果。

## 维护建议

- 修改模型字段后，需要确认 `utils.InitDB` 中的自动迁移是否满足预期。
- 对外响应不要直接返回模型结构，优先通过 DTO 控制字段暴露。
- 涉及枚举值或状态流转时，同步检查 service、handler、测试和文档。
