# utils 目录

`utils` 目录保存多个模块共用的基础能力，包括数据库初始化、JWT、密码哈希、日志和 HTTP 响应封装。

## 文件说明

- `database.go`：SQLite 连接、GORM 配置、模型迁移和默认管理员初始化。
- `hash.go`：密码哈希和密码校验。
- `jwt.go`：JWT 生成、解析、黑名单和测试密钥设置。
- `logger.go`：应用日志初始化。
- `logger_helpers.go`：日志辅助方法。
- `response.go`：统一 HTTP 响应结构和快捷返回函数。

## 维护建议

- 修改数据库初始化逻辑时，确认本地启动、测试初始化和容器运行都能正常工作。
- 修改 JWT 字段或过期策略后，同步检查中间件、用户接口和测试。
- 响应格式调整会影响所有 API 调用方，需要同步更新接口文档。
